Installation
------------

The preferred way to install this extension is through [composer](http://getcomposer.org/download/).

Either run

```
php composer.phar require --prefer-dist yiisoft/yii2-bootstrap4
```

or add

```
"yiisoft/yii2-bootstrap4": "~1.0.0"
```
